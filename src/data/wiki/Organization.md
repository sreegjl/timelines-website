# Organization

Tags and groups help keep larger timelines manageable.

## Tags

Tags are labels that can be applied to events, spans, and eras. They can represent anything useful for your project: topics, categories, people, factions, locations, or any other dimension you want to track across elements. They appear in the sidebar's Tags tab and in the quick tag strip at the top of the Timeline tab.

For each tag you can:
- Set a color, shown as a dot on the element and in the tag strip
- **Spotlight** - highlight only elements with that tag on the canvas
- **Hide** - remove elements with that tag from the canvas entirely
- **Use as Label** - show the tag as a label on elements in the timeline canvas

Tags are created on the fly when added to an element.

## Groups

Groups are display layers that organize elements into separate visual lanes on the canvas. They can represent parallel storylines, characters, regions, thematic threads, or any set of elements you want to visually separate. Every event and span belongs to a group. Eras do not support groups.

From the Groups tab in the sidebar you can:
- Add, rename, and delete groups
- Reorder groups by dragging
- Toggle a group's visibility to show or hide all its elements at once
- Enable a color band to visually mark the group's lane on the canvas

When creating or editing an element, you can assign it to any existing group or create a new one on the fly from the group dropdown.
